# Sami Saif Portfolio Website

This project is a personal portfolio website created for a professional video editor. The website showcases creative work across multiple categories, including reels, podcasts, motion graphics, and explainer videos. It provides visitors with an engaging way to explore projects, learn about the editor's experience, and view featured content.

The portfolio is designed with a clean and modern interface, smooth navigation, and a responsive layout that works across different devices.

## Setup and Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/sami-saif-portfolio.git
cd sami-saif-portfolio
```

Open the website locally:

```bash
index.htm
```

Or run a local server:

```bash
python -m http.server
```

Then visit:

```text
http://localhost:8000
```
